A web-based healthcare platform developed as a graduation project to improve health risk awareness and streamline hospital management tasks. 

## Key Features
* **Web Platform:** Built a functional, responsive web-based framework using the **Django** web framework to handle user and data workflows.
* **Database Management:** Structured and managed relational data tracking using **MySQL** for backend records.
* **System Modules:** Designed and integrated modules for first-aid awareness tutorials, chronic disease risk , and overall hospital system management.

## Tech Stack
* **Framework:** Django (Python)
* **Database:** MySQL
* **Frontend:** HTML, CSS, JavaScript
