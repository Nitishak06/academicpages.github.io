A project focused on exploring text data and understanding emotional trends in datasets using Natural Language Processing (NLP) and Python. 

## Key Features
* **Exploratory Data Analysis:** Conducted data cleaning and analysis using **Pandas** to visualize emotional shifts and distribution patterns in text.
* **Data Visualization:** Utilized **Seaborn** and **Matplotlib** to generate charts and graphs showing how different emotional trends are distributed.
* **NLP Concepts:** Explored basic text processing workflows to handle, filter, and structure youth sentiment data.

## Tech Stack
* **Languages:** Python
* **Frameworks & Libraries:** Flask, Transformers (Hugging Face), DistilBERT, Pandas, NumPy, Seaborn, Matplotlib, NLTK
